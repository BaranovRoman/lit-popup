# LitPopup

## A customizable, modular, lightweight popup plugin with no dependencies (*Work in progress*).

## Why?
I needed full control over popup animation and popup lifecycle.

## Installation

### NPM

Install it from NPM:
`npm i lit-popup`

## API

### Hooks / lifecycle callbacks

Hook | Parameters

onOpen | none
onOpenComplete | none
